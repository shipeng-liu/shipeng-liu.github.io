# HRT-PR Overcooked Project Page

This is a dedicated subpage for the **HRT-PR Overcooked** research project, showcasing the Human-Robot Teaming Framework with Proactive Reactive Feedback.

## Project Overview

**Effect of Adaptive Communication Support on LLM-powered Human-AI Collaboration**

This project investigates adaptive communication strategies for LLM-powered human-AI collaboration, developing frameworks for dynamic communication support in collaborative scenarios.

 ## Page Structure
 
 - `index.html` - Main project page with detailed information
 - `assets/` - Project images and figures
 - `placeholder-images.html` - Reference for placeholder images used in the project
 - `README.md` - This documentation file
