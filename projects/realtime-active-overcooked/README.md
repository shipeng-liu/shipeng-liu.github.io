# Towards Real-time Adaptation of Embodied Agent in Human-Robot Collaboration

## Project Overview

This project introduces a benchmark for evaluating real-time adaptation capabilities of LLM-based embodied agents in human-robot collaboration scenarios, along with the MonTA (Monitor-then-Adapt) framework for enabling proactive, real-time adaptation.

## Key Contributions

- **Enhanced Overcooked-AI Benchmark**: A fine-grained benchmark with 22 layouts of varying complexity to assess agents' proactive adaptability and temporal responsiveness
- **MonTA Framework**: A hierarchical approach inspired by human cognitive studies, using fast monitoring LLMs and slower, high-accuracy adapters
- **Real-time Adaptation**: Enables embodied agents to make proactive, real-time, and reasonable adaptations during collaboration
- **Human-AI Collaboration**: Demonstrates effective communication and instruction capabilities for human-robot teaming

## Research Focus

The work addresses two key challenges in human-robot collaboration:
1. **Substantial LLM inference latency** that impedes instantaneous reasoning
2. **Need for clear rationale** in adaptation strategies and human instructions

## Results

- MonTA outperforms baseline agents across all tested layouts
- Achieves 75% reasonability and consistency in human expert evaluations
- Demonstrates significant performance improvements in low-teaming-fluency scenarios
- Enables real-time adaptation without compromising collaboration experience
